# Imagify – Text to Image Generation Web App

Imagify is a full-stack web application that generates AI-based images from user-provided text prompts.  
The project uses the **ClipDrop Text-to-Image API** to convert text into images through a simple and clean user interface.

---

## Features

- Generate images from text prompts using AI  
- Secure user authentication  
- Credit-based image generation system  
- Clean and responsive user interface  
- Real-time image rendering  

---

## Tech Stack

**Frontend**
- React.js
- JavaScript
- HTML
- CSS

**Backend**
- Node.js
- Express.js

**API**
- ClipDrop Text-to-Image API

---

## ⚙️ How It Works

1. User logs in to the application  
2. User enters a text prompt  
3. The backend sends the prompt to the ClipDrop API  
4. AI-generated image is returned and displayed  
5. Image generation is controlled using a credit-based system
